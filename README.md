# 주식 데이터 파이프라인 구축

Project overview

해당 프로젝트는 코스피/코스닥 및 각종 금융 지표를 수집하여, dashboard 구축 및 api serving이 목적인 프로젝트입니다. \
수집한 다양한 금융정보 data를 적재하여, 주가정보 dashboard 및 의사결정에 도움이 되는 다양한 지표를 구현해봅니다. \
적재된 data를 외부로 부터 제공하는 api 를 빌드 해보는 걸 목표로 가지고있습니다.
